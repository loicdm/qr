# qr

## build instructions :
Run in project root directory :
 - for debug :
    ```console
    cargo build
    ```
 -  for release :
    ```console
    cargo build -r
    ```

*(generated binaries are located in the target subdirectory)*

## run instruction :

- to generate a svg file :
  ```console
  qrs text image.svg
  ```
- to generate a png file :
  ```console
  qrp text image.png
  ```
